Farm Management System

project Overview

The Farm Management System is a Salesforce-based application developed to help farmers manage their farm activities efficiently.
This system stores and manages important information such as farmer details, farm information, crop details, and buyer data.

The main goal of this project is to simplify farm data management and improve communication between farmers and buyers.

---

 Objectives

- To manage farmer details in a centralized system
- To store farm and crop information easily
- To maintain buyer details for selling crops
- To provide a simple and user-friendly interface

---

 Modules in the System

1. Farmer Module

This module stores farmer information such as:

- Farmer Name
- Contact Details
- Address
- Farm Information

2. Farm Module

This module manages farm-related details like:

- Farm Name
- Location
- Farm Size
- Soil Type

3. Crop Module

This module stores crop information such as:

- Crop Name
- Crop Type
- Season
- Quantity

4. Buyer Module

This module maintains buyer information like:

- Buyer Name
- Contact Details
- Crop Interested

---

 Technology Used

- Salesforce Platform
- Apex
- Lightning App Builder
- Object Relationships
